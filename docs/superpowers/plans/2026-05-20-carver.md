# `carver` Engine Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the `carver` engine end-to-end via the Engine SDLC ADD path — Carver-method (Robert Carver, *Systematic Trading* 2015) multi-forecast volatility-targeted monthly equity portfolio — registered initially in `LifecycleState.LAB`, with a SP-B-compliant `LAB_TARGET` so `python -m ops.lab --target-engine carver` works day-zero.

**Architecture:** 5-plug structure mirroring `momentum/` (closest topology — batch monthly, day-market, no per-name stops, batch-gate-routed risk). Three orthogonal per-instrument forecasts (EWMAC trend / 12m value proxy / 20d Bollinger Z) are independently scaled (rolling-abs-forecast→10, capped ±20) then equal-weight combined × Forecast Diversification Multiplier (FDM, bounded 1.0–2.5), then sized via `position_notional = (combined_forecast/10) × (daily_cash_vol_target / instrument_daily_cash_vol)` (Half-Kelly, default 25% annualized vol target). Cadence `MONTHLY_FIRST_TRADING_DAY` enforced by `tpcore.engine_profile.should_fire`; orders are day-market through `tpcore.risk.batch_gate.gate_batch_order`. Engine prefix `cv_`. The build PR does NOT need to clear the DSR/credibility gate (LAB lifecycle is safe-by-construction); LAB → PAPER is a separate, automated ECR-MODIFY once `python -m ops.lab --target-engine carver` produces a dossier that clears DSR ≥ 0.95 ∧ cred ≥ 60.

**Tech Stack:** Python 3.11, pydantic v2 (frozen `BaseModel` + `ConfigDict`), structlog, asyncpg, pandas/numpy (backtest panel math), `tpcore/` shared library (every concern reused — see §12 "Reuse-vs-new" in the spec), pytest + pytest-asyncio + pytest-xdist (`xdist_group("ops_shadow")` where any test touches `sys.modules['ops']`), ruff, `tpcore.scripts.check_imports`, `scripts/gen_engine_manifest.py`.

```python
# ── Carver-method constants (see spec §4) ──────────────────────────────
```

```python
def _ewmac_forecast(prices: pd.Series, fast: int, slow: int, vol_24m: float) -> float:
    """Carver EWMAC: (EWMA_fast - EWMA_slow) / σ_24m. Returns RAW forecast."""
    if vol_24m <= 0 or len(prices) < slow + 50:
        return float("nan")
    f = prices.ewm(span=fast, adjust=False).mean().iloc[-1]
    s = prices.ewm(span=slow, adjust=False).mean().iloc[-1]
    return float((f - s) / vol_24m)


def _value_proxy_forecast(prices: pd.Series, lookback_months: int) -> float:
    """Equity-substrate value proxy: −(12-1 month total return) / σ_24m
    so that 'cheap relative to its own 12-month mean' is a POSITIVE forecast."""
    lookback_days = lookback_months * 21
    if len(prices) < lookback_days + 21:
        return float("nan")
    p_now = prices.iloc[-21]  # skip last month
    p_then = prices.iloc[-lookback_days - 21]
    if p_then <= 0:
        return float("nan")
    raw_return = (p_now / p_then) - 1.0
    vol_24m = float(prices.pct_change().tail(504).std(ddof=1))
    if vol_24m <= 0:
        return float("nan")
    return float(-raw_return / vol_24m)  # sign-flipped: cheap → positive


def _bollinger_z_forecast(prices: pd.Series, window: int) -> float:
    """Mean-reversion Z-score: (EMA_window - close) / σ_window.
    Positive when price below its EMA (cheap)."""
    if len(prices) < window + 1:
        return float("nan")
    ema = prices.ewm(span=window, adjust=False).mean().iloc[-1]
    sig = float(prices.tail(window).std(ddof=1))
    if sig <= 0:
        return float("nan")
    return float((ema - prices.iloc[-1]) / sig)


def _scale_forecast(raw: float, scaling_constant: float) -> float:
    """Multiply raw forecast by per-rule calibration constant so rolling
    abs-mean ≈ FORECAST_TARGET_ABS=10. Default constants below are
    calibrated empirically on T1+T2 across 2018-2025 — see test_models
    for the in-range assertion."""
    return raw * scaling_constant


def _compute_idm(scaled_forecasts: list[float], correlation_matrix: np.ndarray | None) -> Decimal:
    """Forecast Diversification Multiplier — Carver chapter 8.
    FDM = √N / √(1ᵀρ1 / N²), bounded [IDM_FLOOR, IDM_CAP]. Falls back
    to IDM_FLOOR (1.0) on cold-start (correlation_matrix is None)."""
    if correlation_matrix is None:
        return IDM_FLOOR
    n = len(scaled_forecasts)
    if n == 0:
        return IDM_FLOOR
    ones = np.ones(n)
    quad = float(ones @ correlation_matrix @ ones) / (n * n)
    if quad <= 0:
        return IDM_FLOOR
    fdm = float(np.sqrt(n) / np.sqrt(quad))
    return max(IDM_FLOOR, min(IDM_CAP, Decimal(str(fdm))))
```
